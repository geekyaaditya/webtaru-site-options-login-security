# Webtaru Site Options and Login Security

**Webtaru Site Options and Login Security** is a powerful WordPress plugin designed to help you manage your site's essential information and security from a single, centralized dashboard.

## Key Features

- **Custom Login URL:** Protect your site from brute-force attacks by changing the default `/wp-login.php` to a custom slug.
- **CAPTCHA Protection:** Integrate Google reCAPTCHA v3 or Cloudflare Turnstile to stop bot logins.
- **Dynamic Business Information:** Manage phone numbers, emails, addresses, and social links via simple settings.
- **Shortcode Powered:** Display your information anywhere using easy-to-use shortcodes like `[wtols_phone]`.
- **SEO Optimized:** Automatically generates Schema.org LocalBusiness JSON-LD for your site.
- **Agency Ready:** White-label the plugin for your clients with a custom menu name.

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin via the 'Plugins' menu.
3. Configure your settings under the **Webtaru Site Options** menu.

## Compliance Note

This plugin is fully compliant with the WordPress.org Plugin Directory Guidelines. It does not contain any arbitrary code execution features and follows strict security practices for data handling and output.
